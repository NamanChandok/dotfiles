# Nobara dotfiles

**OS:** Nobara Linux

**DM:** GNOME

**Browser:** Zen

**IDE:** Zeditor

**Terminal:** Ghostty

**Shell:** ZSH

**Icon Theme:** ![WhiteSur Alternate](https://github.com/vinceliuice/WhiteSur-icon-theme)

**Font:** Jetbrains Mono

**List of GNOME Extensions:**
- AppIndicator and KStatusNotification Support
- BlurMyShell
- Caffeine
- Clipboard Indicator
- Color Picker
- Dash to Dock
- Logo Menu
- Media Controls
- moveclock
- Open bar
- Tiling Shell
- Window title is back
