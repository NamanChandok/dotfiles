# Nobara dotfiles

**OS:** Nobara Linux

**DM:** GNOME

**Browser:** Zen

**Terminal:** Ghostty

**Shell:** ZSH

List of GNOME Extensions:
- AppIndicator and KStatusNotification Support
- BlurMyShell
- Caffeine
- Clipboard Indicator
- Dash to Dock
- Just Perfection
- Logo Menu
- Media Controls
- Tiling Shell
