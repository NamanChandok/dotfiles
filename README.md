# Nobara dotfiles

**OS:** Nobara Linux

**DM:** GNOME

**Browser:** Zen

**IDE:** Zeditor

**Terminal:** Ghostty

**Shell:** ZSH

**List of GNOME Extensions:**
- AppIndicator and KStatusNotification Support
- BlurMyShell
- Caffeine
- Clipboard Indicator
- Color Picker
- Dash to Dock
- Just Perfection
- Logo Menu
- Media Controls
- Tiling Shell
